Frontend Interaction:
1. Provide file path
2. Select modules
3. Provide export path
4. Receives new file


Backend Interaction:
1. Get’s the file path
   1. If the file is invalid, throw an exception and ask again
2. Load the provided file
   1. Get metadata
   2. Read to vector<double>
3. Get selected modules
   1. Give priority to cleaning modules
4. Upon export
   1. Request name
   2. Apply selected modules
   3. Generate the file with the given name, using the original path.
