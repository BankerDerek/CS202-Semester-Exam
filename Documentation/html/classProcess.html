Detailed Description

Processor for wave file data.

Process class acts as the processor for all the data. The class inherets all the properties of the classes that declare and define the functions for filtering or modifying the given data. To use, create an object assigned to the "Process" class and then you are free to use any function in any of the data modifying classes.
