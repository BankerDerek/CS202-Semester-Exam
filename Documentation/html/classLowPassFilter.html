Detailed Description

Class that contains a low pass filter.

The class contains a function that conducts running a vector of data through a low pass filter. It does this by multiplying the original vector by a slightly delayed version of that vector. It then runs the resulting vector through a gain and then recombines the modified vector with the original vector.
